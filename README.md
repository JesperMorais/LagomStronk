## Features

- **Workout Tracking** — Log exercises, sets, reps, and weights
- **1RM Calculator** — Calculate your one-rep max
- **Progress History** — View past workouts and PRs
- **Offline Support** — Works without internet (PWA)
- **Android App** — Available via Capacitor build

## Quick Start

```bash
npm install
npm start
```

## Android Build

```bash
npm run build:android
```

## Tech Stack

`PWA` `Capacitor` `Vanilla JS` `Service Worker`

## Goal
Create custom numeric keypad for weight/reps input matching inspiration/numpad.jpg design.

## Context
User wants numpad with:
- 0-9 digits, decimal, backspace
- +/- steppers (2.5kg for weight, 1 for reps)
- NEXT button to advance to next field
- Dark theme, clean layout
- Keyboard toggle to fallback to system keyboard

## Tasks

<task id="1">
Add numpad HTML to index.html (hidden by default):
- Fixed position at bottom of screen
- 4x4 grid: digits, decimal, backspace
- Plus/minus buttons on side
- NEXT button spanning bottom
- Keyboard toggle button
</task>

<task id="2">
Add numpad CSS to style.css:
- Dark background matching app theme
- Large touch targets (min 48px)
- Mint accent for NEXT button
- z-index: 1001 (above modals)
- Slide-up animation
</task>

<task id="3">
Create numpad functions in app.js:
- showNumpad(inputElement, options) - display and focus
- hideNumpad() - hide with animation
- Handle digit input, decimal, backspace
- Handle +/- with configurable step
- NEXT advances to next input or closes
</task>

<task id="4">
Integrate with inline set inputs:
- Weight inputs show numpad on focus
- Reps inputs show numpad on focus
- Set readonly attribute to prevent system keyboard
- NEXT moves: reps -> weight -> next set's reps
</task>

<task id="5">
TEST: Tap a weight input, verify:
- Numpad appears with slide-up animation
- Can enter digits (e.g., 62.5)
- +/- adds/subtracts 2.5
- NEXT moves to reps input
- Tapping outside closes numpad
</task>
